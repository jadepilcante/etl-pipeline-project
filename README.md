# Energy Data Pipeline

Pipeline ETL con Pandas que automatiza la transformación de datos energéticos.

## Descripción

Los analistas perdían días cada trimestre limpiando manualmente datos de ventas y capacidad. Este pipeline automatiza el proceso, reduciendo el tiempo de procesamiento al ejecutarse mensualmente y entregar los datos listos para el análisis.

## Datos

- `electricity_sales.csv` - Ventas de electricidad
- `electricity_capability_nested.json` - Capacidad energética anidada

## Tecnologías

- Python
- Pandas
